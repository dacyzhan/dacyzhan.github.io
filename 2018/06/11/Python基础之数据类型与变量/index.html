1.Python 3.0 中的注释:

Python允许用 # 单行注释

Python允许用’’’…’’’的格式表示多行内容

‘’’ 代码块注释’’’

“”” 代码块注释 “””

值得注意的是 单行注释与下方代码块是否对其无关。但是 块注释中必须在方法外或者 与下方代码块对其，否则代码将会从该代码块中开始执行。

print('''
lin1
lin2
lin3''')

2.转义符的使用

\’ 表示 ‘ ,斜杠为转义符

print('I\'am ok.')

输出结果如下图所示：

转义字符\可以转义很多字符，比如\n表示换行，\t表示制表符，字符\本身也要转义，所以\表示的字符就是\

print('\\\t\\')

print('\\\n\\')

输出结果为下图所示：

Python允许用r’’表示’’内部的字符串默认不转义

print(r'\\')

输出结果如下图所示：
