Python开发之环境搭建

Python 开发准备

1. Python 安装以及环境配置

1.1 Python 下载与安装

Python 的下载

python 安装与其他软件没任何区别， 官网下载地址:

https://www.python.org/
打开官网见下图

下载直接按照下图的步骤下载即可。

当然上面下载的是最新的版本当前最新为 3.6.5 。 有些同学可能想学习之前的历史版本可以按照下图 中根据自己的操作系统不同进行选择下载。


目前我这里选择的是 Windows 版的64 位 的 3.6.4 的版本进行学习。

1.1.2 Python 的安装

其实Python 安装和其他软件没什么区别。 Windows 中安装就直接双击 安装文件进行安装即可。不过要修改下指定的安装路径，并记录，后面配置环境时需要。
这个我安装完之后的界面，如下图所示。

到此就安装成功了。

1.1.3 Python 环境配置 （windows 下配置方式）

python 环境变量配置也相当简单的。点击 “我的电脑” 右键属性 ，然后安装下图步骤 修改 环境变量 “path” ,将光标移动到变量值的最后面，然后添加 分号，再将安装 python 目录添加进去即可。

检验 pyhon 安装是否成功。打开 CMD 窗口，输入 python , 如返回 python的版本号，则说明安装成功。 否则按照之前的步骤重新检查下。

至此，恭喜你，你的Python环境已经成功安装完成，可以正式开始进入 Python 的学习与开发啦。
