回首过去，展望未来

#回首过去，展望未来

###踏上寻求改变人生之路。

###本博客后期会以python系列为主。

####回顾以前

在这全新的开始之前，我首先介绍下自己， 到现在 18 年 6 月份，刚好毕业整整三年了，一直从事着 java 开发的工作。三年，说长不长，说短也不短。期间有去自学过 Android 开发的基础知识，想转去做 Android 但是后面换工作面试了几家后就心虚了，最终还是找了个 java 开发的工作，Android 开发的梦想也就不了了之 。

目前在一家传统企业做 java 开发，公司规模不大，应该算是中小型吧，也会经常焦虑不安，想想以后的发展之路会觉得挺迷茫。其实迷茫的主要原因是对自身能力的否定，常常会觉得自己技术挺水。工作的三年里，java 的技术水平当然也有一定提升，但是和那些技术大神比起来还是相差甚远。 其实我挺早之前就关注张哥了，只是总找不到一个写作的切入点。看着张哥一步步走过来，确认挺厉害和佩服的，然而我却还是止步不前。深深的自责与愧疚呀。

####展望未来

拥抱 python ，跟上时代潮流。只要开始努力，任何时候都不会晚。立个 flag ,从现在开始，我会持续学习 python 。 其实之前项目不太忙的时候学过一段时间 python 的基础知识，但是几个月不用又忘的差不多了。 学以致用，温故而知新，这句话太有道理了。

总结一下，学习新知识必须做到以下几点：
1.学习新知识必须做笔记。
2.多实践才会有更好的认识。
3.学习完最好用文章的形式记录下来，一方面可以共享给需要的人，另一方面也方便自己回顾知识。
4.用小项目去规整和消化学过的知识。
5.上传自己的小项目到开源社区，提升自己的个人影响力。

其实我开通公号的目的很简单，就是想记录下自己的学习路径，多多写作，提升记录的语言组织能力。 很多时候想到和用语言表达出来是两回事。众所周知，程序员工作本就单调，有时候调试程序可能一整天都不会与别人沟通。 沟通能力普遍偏弱，所以多写作才会提升记录的语言组织能力，弥补工作空缺。

最后，希望大家能够一起进步，相信大家都是最优秀的人，早日赢取白富美，走上人生巅峰。

有兴趣的同学的可以关注一下，让我们共同成长。
